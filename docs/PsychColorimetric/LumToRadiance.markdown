 \[radiance, radianceS\] =...
    LumToRadiance\(relativeSpectrum, relativeSpectrumS, luminance, \[photopic\]\)

 Convert luminance in photopic cd/m2 to radiance, given relative spectrum
 of the source.

 Variable photopic can take on values:
        'Photopic' \(Default\)
   'JuddVos'
   'Scotopic'
