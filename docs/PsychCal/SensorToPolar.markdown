 \[pol\] = SensorToPolar\(sensor\)

 Converts from sensor rectangular coordinates to polar
 coordinates.

 Polar coordinates are defined as radius, azimuth, and elevation.

 See also PolarToSensor, SensorToCyl, CylToSensor.
