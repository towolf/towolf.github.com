PR670rawspd - Takes an spd measurement and returns the results.

Syntax:
readStr = PR655rawspd(timeout)

Input:
timeout (scalar) - Timeout period in seconds.

Output:
readStr (1xN char) - The raw character array resulting from a
    measurement.
