s=SizeOfRect\(rect\);
Accepts a Psychtoolbox rect \[left, top, right, bottom\] and returns the
size \[rows columns\] of a MATLAB array \(i.e. image\) just big enough to
hold all the pixels.
See also RectOfMatrix, RectHeight, RectWidth, PsychRects.
