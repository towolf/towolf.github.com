trolands = PowerToTrolands(wl,power,S\_vLambda,vLambda)

Convert from power (in watts/deg-2) to trolands for a
monochromatic light.  Wavelength specified in nm,
vLambda should be CIE photopic luminosity function.

See W&S, p. 103.
